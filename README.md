# CS50P - Introduction to Programming with Python

This repository contains my solutions for **CS50's Introduction to Programming with Python (CS50P)**, including all problem sets and my final project.

## About the Course

CS50P is an introduction to programming using Python. It covers reading and writing code, debugging, testing, and building practical programs.

Main topics include:

- Functions, arguments, and return values
- Variables and data types
- Conditionals and Boolean expressions
- Loops
- Exceptions and debugging
- Unit testing
- Third-party libraries
- Regular expressions
- Classes, objects, methods, and properties
- File input/output

## Final Project

### Wuzzuf Job Scraper

Final Project Repository: [CS50P-Final-Project](https://github.com/kmashaly24/CS50P-Final-Project)

The final project is a Python script that scrapes job listings from the Wuzzuf website and saves results into a CSV file.

#### Features

- Scrapes job data using `requests` and `BeautifulSoup`
- Extracts job title, company name, location, post time, job type, site type, and skills
- Exports data to `wuzzuf.csv`
- Uses `pandas` to display results in table format

#### Requirements

- Python 3.x
- `pandas`
- `requests`
- `beautifulsoup4`

#### Usage

```bash
python project.py
```

Then enter a Wuzzuf search URL when prompted.

#### Testing

Unit tests are written with `pytest` in `test_project.py`.

Run tests with:

```bash
pytest test_project.py
```
